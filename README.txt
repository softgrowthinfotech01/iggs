Indira Gandhi School Chandrapur Website

Use in XAMPP:
1. Copy this folder into htdocs.
2. Open: http://localhost/indira_gandhi_school_website/
3. Edit header.php and footer.php once. Changes will show on all pages.
4. Replace SVG images in images folder with real JPG/PNG files if needed.
5. Add your school video as videos/school-tour.mp4.
